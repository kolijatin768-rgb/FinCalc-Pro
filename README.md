Project: FinCalc Pro


FinCalc Pro is a modern, responsive financial and scientific calculator designed to provide real-time currency conversion and advanced mathematical functions in a beautiful, glassmorphism-inspired interface.

🚀 Features

* Scientific Calculator: Supports advanced mathematical functions including `sin`, `cos`, `tan`, and `log`.
* Live Currency Conversion: Integrated with the Frankfurter API to provide real-time exchange rates for multiple currencies (USD, INR, EUR, GBP, JPY, AUD, CAD).
* Glassmorphism UI: A sleek, modern design with a glass-effect aesthetic that adjusts seamlessly between light and dark themes.
* Progressive Web App (PWA): Installable directly to your device for offline access and native-like performance.
* Responsive Design: Optimized for mobile and desktop viewing.

🛠 Tech Stack

* Frontend: Vanilla JavaScript, HTML5, CSS3.
* PWA: Service Workers for offline caching and Web Manifest for installation.
* API: Frankfurter API for live currency exchange data.
* Analytics: Vercel Insights for performance and usage monitoring.

📲 How to Install (PWA)

1. Open the application in your browser.
2. Click the Download icon (📥) located in the top-right header of the calculator.
3. Follow your browser's prompt to install the app on your device or home screen.

💻 How to Use

* Standard Math: Select "Standard Math" from the dropdown to perform basic and scientific arithmetic.
* Currency Conversion: Select "Currency Converter" to see the live conversion row.
* Keyboard Support: You can use your physical keyboard to type numbers, operators (`+`, `-`, `*`, `/`), and perform calculations by pressing `Enter`.
